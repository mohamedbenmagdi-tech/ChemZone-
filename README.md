# ChemZone — منصة د. محمد مجدي لتدريس الكيمياء

موقع ثابت (HTML/CSS/JS) جاهز للرفع على GitHub ونشره مجانًا عن طريق **GitHub Pages**.

## الملفات
- `index.html` — الصفحة الرئيسية (تعريفية فقط، بدون عرض الكورسات)
- `login.html` — تسجيل الدخول
- `register.html` — إنشاء حساب
- `dashboard.html` — لوحة تحكم الطالب (بعد الدخول)
- `courses.html` — صفحة الكورسات: تبويب "الثانوية العامة" (5 أبواب) وتبويب "البكالوريا"
- `assets/style.css` — التنسيق العام للموقع
- `assets/teacher.png` — صورة المعلم

> ملحوظة: تسجيل الدخول والتسجيل في هذه النسخة **تجريبي (Demo)** بدون سيرفر حقيقي — بيدخلك على لوحة التحكم مباشرة لعرض الشكل والتجربة. لو عايز حسابات حقيقية وحفظ بيانات الطلاب، هتحتاج تربط الموقع بخدمة مجانية زي Firebase أو Supabase لاحقًا.

## إزاي ترفعه على GitHub وتنشره مجانًا

1. اعمل حساب على [github.com](https://github.com) لو مفيش عندك.
2. اعمل Repository جديد (مثلاً اسمه `chemzone`)، خليه **Public**.
3. ارفع كل ملفات المجلد ده (`index.html`, `login.html`, ... و مجلد `assets`) داخل الـ Repository.
4. من إعدادات الـ Repository: **Settings → Pages**.
5. من **Source** اختار الفرع `main` والمجلد `/ (root)`، ثم اضغط **Save**.
6. بعد دقيقة أو اتنين، هيديك GitHub رابط شبه:
   `https://username.github.io/chemzone/`
   وده رابط موقعك المجاني.
